# Ventauri Esports
A merchandise and team website for Ventauri Esports, a competitive Formula 1 Esports racing team.

## About
Ventauri Esports is dedicated to competing at the highest level of Formula 1 Esports racing. This website serves as the official platform for team information, merchandise sales, and fan engagement.

## Features
### Team Information
- Team roster and driver profiles
- Race results and championship standings
- News and updates from the racing circuit
### Merchandise Store
- Official team apparel and accessories
- Secure online shopping experience
- User account management for order tracking
- Wishlist functionality for favorite items
### User Experience
- User registration and authentication
- Personal profile management
- Order history and tracking
- Address and payment method management
- Product reviews and ratings
### Administrative Features
- Comprehensive admin dashboard
- Product catalog management
- Order processing and fulfillment
- User management and support
- Promotional coupon system
## Technology Stack
- Frontend : React with TypeScript
- Backend : Go with Gin framework
- Database : MariaDB
- Authentication : Google oAuth
## Project Status
This project is currently in active development. The core backend infrastructure has been implemented, with ongoing optimization of data management operations across all system components.
